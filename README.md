## Overview

Security Log Analyzer is a simple Python tool designed to analyze log files and detect suspicious login attempts.  
It helps identify potential brute-force attacks by detecting multiple failed login attempts from the same IP address.

This project demonstrates basic **security log analysis**, which is commonly used in **Security Operations Centers (SOC)**.

---

## Features

- Detect failed login attempts
- Identify suspicious IP addresses
- Generate a simple security report
- Lightweight and easy to use

## Technologies Used

- Python
- Regular Expressions (re module)
- Collections module
